
# Distributed API Orchestration Platform

## Overview
This project orchestrates multiple APIs using asynchronous workflows, retries, logging, authentication, and monitoring.

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Chart.js
- Backend: Node.js, Express
- Architecture: Distributed workflow orchestration

## Features
- Trigger multi-step API workflows
- Centralized orchestration backend
- Monitoring dashboard
- Internship-ready clean architecture

## How to Run

### Full Application
```
cd backend
npm install
npm start
```
Then open http://localhost:5000 in your browser to access the application.

## Use Case
Automates chained API calls like payment flows, microservices orchestration, and data pipelines.
